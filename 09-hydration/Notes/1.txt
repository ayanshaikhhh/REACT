Hydration Error in React JS usually tab aata hai jab server-side rendered (SSR) HTML aur client-side React ka HTML match nahi karta.
Ye error mostly Next.js, React 18 (SSR), ya hydrateRoot() use karne par aata hai.

Window / document use karna server-side par
